Certification / accreditation marks for the About page.

The page ships with clean vector (SVG) versions of each mark, so nothing 404s
and every logo stays crisp at any size:

  asa-member.svg     -> ASA Member (Aviation Suppliers Association)
  faa-certified.svg  -> FAA AC 00-56B Certified Company seal
  inc-500.svg        -> Inc. 500 America's Fastest-Growing Private Companies
  dla-cage.svg       -> Defense Logistics Agency (CAGE Code 6RE77)
  nbaa-member.svg    -> Member of NBAA

These are faithful, recognizable recreations for the mockup/preview. To swap in
the exact official artwork later, drop the vendor-supplied file in this folder
and update the matching <img src="logos/..."> in ../about.html. PNG or SVG both
work; the tile scales to max-height 104px. Keep a transparent or white
background so it reads on the white logo tiles.
